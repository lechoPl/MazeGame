MazeGame
========

Simple console game in C#

Most of variables in code are in Polish

About game:
In this game we need to find exit from maze.
Maze is generated randomly.

	D - door to next room
	H - special door to finsh game
	K - key to door 
	I - special key to open H door
	$ - money :)
	☻ - enemy
	☺ - player
	♦ - wall

Controls:
	w, s, a, d 	- move
	p 			- pick object
	o 			- open door
	l 			- use key
	esc			- exit
